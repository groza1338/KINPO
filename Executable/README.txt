# TheCalculationOfWaterVolumeBetweenWalls
## Программа для вычисления объема воды между стенами


## Описание

Программа предназначена для осуществления расчета объема воды который может быть удержан между стен.

Программа должна принимать высоту стенок, рассчитывать количество воды, которое может быть удержано над каждой из стенок и возвращать общее число воды, которое может быть удержано над стенками и схему стен и воды.


## Использование


Для запуска программы используйте следующую команду:
Gvozdkov_run.exe <путь_к_входному_файлу> <путь_к_выходному_файлу>

Пример команды:
```shell
.\Gvozdkov_run.exe ..\Tests\Input\input01.txt .\output.txt

```



Для проверки работы функций программы используйте следующую команду (перед этим нужно убедиться, что [Gvozdkov_test.exe](Gvozdkov_test.exe) находится рядом):
```shell
.\Gvozdkov_run.exe --test
```

## Входные данные


Программа принимает текстовый файл с расширением `.txt`, содержащий высоту стенок. Высота представляет из себя число в диапазоне [0..4294967295]. Высоты разделяются любым пробельным символом:

Пример входного файла:
```markdown
4 0 4
```

## Выходные данные


Выходной файл также является текстовым файлом с расширением `.txt`, содержащий в первой строке объем воды, удержанной между стенок и строками ниже схему стен и воды.

Пример выходного файла:
4
```markdown
# ~ #
# ~ #
# ~ #
# ~ #
```


## Требования


- Windows-совместимая система.
- Файл входных данных должен быть корректно отформатирован.
- Входные параметры должны быть числами.
- Не должны быть числа вне диапазона [0..4294967295].
